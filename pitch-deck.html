Grading System

+1.0 to −1.0 per principle. A+ to F.

Each of 8 principles scores +1.0 (Exemplary) / +0.5 (Acceptable) / −0.5 (Concerning) / −1.0 (Violation). Final score is the mean — grades map A+ (≥0.875) down to F.

GradeScore RangeWhat it means
A+≥ 0.875Exemplary across all principles
A≥ 0.625Strong, high-signal community contribution
B≥ 0.375Good — minor room for improvement
C≥ 0.125Average — some concerning elements
D≥ −0.125Below average — notable violations
F< −0.125Harmful — violates multiple principles
Architecture

Built on W&B + LangGraph + Redis

🔗

LangGraph Supervisor

Orchestrates the 4-agent pipeline. State machine with typed edges — scrape → blueprint → index → grade → done. Each agent is a LangGraph node with weave tracing.

🔬

WandB Weave + LLM

All agent calls traced via @weave.op. LLM inference through WandB's OpenAI-compatible endpoint — Qwen3-14B-Instruct for blueprint + grading.

🧠

Redis VL + RAG

Comments embedded with sentence-transformers (384-dim HNSW). HumaneBench KB stored in Redis VL. Grader uses RAG to pull relevant principle examples.

💬

CopilotKit AI Chat

Three in-context AI panels — Blueprint AI, Vibe Audit AI, Prompt Chip AI — each wired to relevant CopilotReadable context. Powered by WandB LLM.
